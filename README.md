# Jasmine File Tree Matcher

A Jasmine matcher for testing if two file trees are identical.

[![CircleCI](https://circleci.com/gh/typescript-plus/jasmine-file-tree-matcher.svg?style=shield)](https://circleci.com/gh/typescript-plus/jasmine-file-tree-matcher)

## Install

```
npm i @typescript-plus/jasmine-file-tree-matcher
```
